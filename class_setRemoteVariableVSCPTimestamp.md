## setRemoteVariableVSCPTimestamp

```c
int setRemoteVariableVSCPTimestamp( const wxString& name, 
                                    uint32_t vscp_timestamp)
```

### Parameters

#### name
Name of remote variable to set value for.

#### vscp_timestamp
VSCP timestamp to write to remote variable.

### Description
