```clike
int setRemoteVariableDate( const wxString& name,

```

### Parameters

#### name
Name of remote variable to set value for.

#### date
VSCP date to write to remote variable.

### Return Value
CANAL_ERROR_SUCCESS on success or an error code on failure. 

### Description
